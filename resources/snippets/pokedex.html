Pokedex

Pokedex project made with Kotlin, executable as an Android APK. This project consumes the Poké API for fetching pokemon info.

Poké API is a RESTful API containing rich information about Pokemons from various pokemon generations, including pokemon status, images, and habilities from different games of Pokemon series. This project gathered some of the main informations of the pokemons, forming a simple Pokedex, where its possible to search pokemons by name and click to see pokemon data.

Libraries:

  • Http Client retrofit
  • Deserialization with GSON
  • Image rendering with Picasso
