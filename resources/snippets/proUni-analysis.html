Data Analysis in ProUni 2018

This project analysed possible correlations in data at ProUni 2018 DataSet. To perform an analysis on a visual manner, different graphs were constructed using python libraries, majorly correlating students grades and scolarships types:

Scolarships:

  • Full scholarship quotas;
  • Full open scholarship;
  • Partial scholarship quotas;
  • Partial open scholarship.

Libraries:

  • Pandas;
  • Numpy;
  • Matplotlib;
  • plotly;
  • Seaborn.
